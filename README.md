# أكاديمية الخبير — El Khabir Academy

موقع هبوط (landing page) لمنتج تعليمي عقاري يهدف إلى توعية المشترين الجزائريين قبل
خوض أول صفقة عقارية. يتضمن الموقع نموذج طلب مع تحديد الولاية والبلدية والتوصيل.

## المنتج

- **دليل "أول صفقة عقارية"** — كتاب ورقي يشرح أساسيات القانون العقاري الجزائري
- **علبة فلاش ذهبية** — محتوى رقمي إضافي (فيديوهات، نماذج، مستندات)

## المميزات

- نموذج طلب ديناميكي مع اختيار الولاية والبلدية (قائمة 58 ولاية + بلدياتها)
- خيارا التوصيل: توصيل إلى المكتب (Stop Desk) أو إلى المنزل (Home Delivery)
- تحديث الأسعار تلقائياً حسب الولاية
- تصميم متجاوب (responsive) لجميع الشاشات
- تدرجات لونية ذهبية/داكنة بتصميم فاخر
- 5 أقسام تفاعلية (أفقية) تعرض محتوى الدورة والمنتجات

## التقنيات

- HTML5 & CSS3 خالص (بدون مكتبات خارجية)
- JavaScript (DOM API) لإدارة النموذج والأسعار
- خطوط Google Fonts: El Messiri + Tajawal
- SVG مدمج للرسومات (الأيقونات، الأفق، الختم)

## التشغيل محلياً

```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve .
```

ثم افتح `http://localhost:8000` في المتصفح.
